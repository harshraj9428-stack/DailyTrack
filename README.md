# 🚀 DailyTrack - AI Routine Tracker

DailyTrack is an AI-powered productivity tool that helps students master their daily routines with real-time efficiency tracking, local AI coaching, and cloud synchronization.

## 🌟 Key Features
- **3-Tier Auth**: Google, Email, and Guest login.
- **AI Coach**: Task breakdown, priority ranking, and weekly performance audits using local LLMs (Ollama).
- **Real-time Sync**: Automatic cloud backup via Google Firebase.
- **Scorecard**: Dynamic grading based on the IIT Patna academic scale.
- **Offline-First**: Instant UI updates with local storage persistence.

## 🚀 Quick Start
1. Ensure **Ollama** is running locally (`ollama serve`).
2. Double-click `run.bat` in the root folder.
3. Access the app at `http://localhost:5500`.
